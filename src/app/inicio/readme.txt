instrucciones paso a paso para

RegistrAPP.

## Requisitos Previos

1.- Descargar e Instalar node.js.
    link para descargar node.js => https://nodejs.org/en/

2.- Instalar Angular CLI,
  - Instala Angular CLI globalmente usando npm. 
    Abrir una terminal => (Ctrl+Shift+ñ) y ejecutar 
    el siguiente comando => npm install -g @angular/cli

3.  Instalar Ionic CLI globalmente usando npm.
  - Instale Ionic CLI globalmente usando npm. 
    Abrir una terminal => (Ctrl+Shift+ñ) y ejecutar 
    el siguiente comando => npm install -g @ionic/cli
    

4. 
   - Instalar las dependencias del proyecto utilizando npm:
  
     ejecuta el siguiente comando en tu terminal => npm install
     

  Ejecutar la Aplicación

1. Iniciar el Servidor de Desarrollo:
   - Ejecuta el siguiente comando para iniciar el servidor de desarrollo:
     
     ionic serve
     ```
   - Esto abrirá la aplicación en su navegador predeterminado en `http://localhost:8100`.

## Notas Adicionales

- **Configuración de Firebase**:
  - Asegúrese de tener configurado Firebase Authentication en su proyecto. Puede que necesite configurar las credenciales de Firebase en el archivo de configuración correspondiente.

- **Problemas Comunes**:
  - Si encuentra problemas al ejecutar la aplicación, 
    asegúrese de que todas las dependencias estén
    correctamente instaladas y que no haya errores en la consola del navegador.
